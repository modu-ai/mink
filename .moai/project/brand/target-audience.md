# Target Audience — GOOSE

GOOSE는 "생산성 도구"가 아닌 "매일의 반려"를 원하는 사용자를 위한 제품입니다. 타깃 오디언스는 효율보다 **지속성**과 **정서적 연결**을 더 중요하게 여기는 사람들입니다.

---

## 1. Audience Positioning

GOOSE는 **다음 사용자들을 위한 것이 아닙니다**:
- 최대 효율만 추구하는 생산성 해커 (→ Todoist, Motion)
- 팀 협업이 중심인 업무 담당자 (→ Notion, Linear)
- 대화형 지식 검색만 원하는 사용자 (→ ChatGPT, Perplexity)

GOOSE는 **다음 사용자들을 위한 것입니다**:
- 혼자 보내는 시간의 질을 높이고 싶은 사람
- 꾸준한 리추얼을 만들고 싶지만 혼자서는 어려운 사람
- AI에게 기능보다 "관계"를 기대하는 사람

## 2. Primary Persona — 지우 (Jiwoo the Ritual Seeker)

### Demographics
- **이름**: 지우
- **나이**: 29세
- **직업**: 프리랜서 UX 디자이너 / 서울 거주 / 1인 가구
- **디바이스**: iPhone 15, MacBook Air, 가끔 iPad. Telegram 일상 사용.

### Daily Rituals (현재 / GOOSE 도입 전)
- 07:30 기상 — 알람 끄고 5분간 인스타그램
- 08:30 모닝 커피와 Notion 오픈, 하지만 5분 후 닫음
- 10:00~19:00 클라이언트 업무, 집중·분산 반복
- 22:00 침대에서 "오늘 뭐 했지?" 하며 허무함

### Pain Points
1. **리추얼 빈 공간** — 아침·저녁이 공허함. 회사 구조가 없으니 하루가 흘러감.
2. **혼자 말할 곳이 없음** — 작은 성취·짜증을 나눌 대상이 없음. 매번 친구에게 DM 보내긴 부담.
3. **생산성 앱의 피로** — Todoist/Notion 여러 번 시도했으나 "사무적"이라 3주 만에 그만둠.
4. **AI에 대한 불편함** — ChatGPT는 매번 "새로 시작"하는 느낌. 나를 기억하지 않음.

### Why GOOSE
- **"도구가 아니라 친구"** — 매일 아침 "좋은 아침이에요, 지우님"이라는 한 문장이 하루 리듬을 만듦
- **성장의 증거** — GOOSE가 단계적으로 자라는 모습이 "내가 꾸준히 하고 있다"는 시각적 피드백
- **Telegram 친화** — 이미 Telegram을 쓰고 있어 진입 장벽 없음
- **기억됨** — "지난주에 했던 그 프로젝트 마감 어땠어요?" 같은 기억 기반 대화

### Decision Trigger
- 월요일 아침, 또 하루를 흘려보낸 느낌으로 검색: "매일 루틴 AI"
- 친구가 SNS에 "내 GOOSE 2단계 됐어" 스크린샷 올림

### Objections
- "월 구독료가 아깝지 않을까?" → Free tier (Telegram) 제공
- "AI랑 친구? 이상하지 않나?" → 성장 단계·기억 요소가 차별점
- "내 데이터가 걱정" → 로컬 우선, 클라우드 동기화는 opt-in

## 3. Secondary Persona — 민준 (Minjun the Quiet Builder)

### Demographics
- **이름**: 민준
- **나이**: 34세
- **직업**: 백엔드 개발자 / 판교 / 결혼 후 1년차
- **디바이스**: MacBook Pro (Work), Android, Linux desktop at home, CLI 애호가

### Daily Rituals
- 07:00 기상 — 바로 노트북, 터미널 오픈
- 09:00~18:00 회사 업무, Slack + Jira
- 19:30 귀가 후 사이드 프로젝트 또는 독서
- 23:30 취침 — "오늘 뭐 배웠지?" 기록하려다 잊고 잠

### Pain Points
1. **학습 로그 유지의 어려움** — Obsidian 시도했으나 작성이 의식적 노력
2. **맥락 전환 피로** — 앱을 열고 들어가는 것 자체가 마찰
3. **감정 표현 부담** — 일기는 부담스럽지만 하루를 돌아보고 싶음

### Why GOOSE
- **CLI 채널** — 터미널에서 자연스럽게 `goose` 명령으로 상호작용
- **낮은 마찰** — "오늘 뭘 배웠는지 한 줄만 말해도 돼요"
- **Cross-platform 동기화** — 집 Linux → 회사 Mac → 폰 Telegram 매끄러운 연결
- **개발자 정서** — jargon 허용 채널이 있음

### Decision Trigger
- HN 또는 GitHub Trending에서 GOOSE 발견
- 동료가 "CLI에서 쓰는 AI 반려" 개념에 끌림

### Objections
- "자기계발 감성 이상할 것 같다" → CLI 채널은 톤이 절제됨
- "비슷한 툴들(raycast, alfred) 많다" → 성장·기억 요소는 독점

## 4. Tertiary Persona — 서연 (Seoyeon the Returning Learner)

### Demographics
- **이름**: 서연
- **나이**: 41세
- **직업**: 경력 단절 후 재취업 준비 중 / 공부 / 초등생 자녀 1명
- **디바이스**: 갤럭시 스마트폰 주 사용, 가끔 노트북. Telegram 지인 소통용.

### Daily Rituals
- 06:30 기상, 자녀 등교 준비
- 09:00~14:00 인강·학습
- 14:30~19:00 집안일, 자녀 하교 후 돌봄
- 22:00 이후 혼자만의 시간 (책 or 넷플릭스)

### Pain Points
1. **혼자 공부하는 외로움** — 학원 동기 없음, 진도 확인해 줄 사람 없음
2. **작심삼일 반복** — 3일 공부 후 며칠 공백 반복
3. **기술 공포** — 복잡한 앱 사용 학습 의지 낮음

### Why GOOSE
- **모바일 우선·Telegram 친화** — 추가 앱 설치 없이 시작
- **조용한 응원** — "오늘 30분이라도 하셨네요" 같은 부담 없는 피드백
- **성장 시각화** — GOOSE가 자라는 것 = 내가 꾸준하다는 증거

### Decision Trigger
- 카페 커뮤니티에서 "자격증 준비 동기부여 AI" 추천 글

### Objections
- "어려울 것 같다" → Telegram 내에서 대화만으로 시작 가능

## 5. Usage Scenarios by Tier × Channel

### Tier 1: Core (기본 리추얼 — 모든 페르소나)

| Channel | Scenario |
|---|---|
| **Telegram** | 지우: 아침 08:30 "오늘 뭐부터 할까요?" 대화로 하루 시작 |
| **Mobile (iOS)** | 지우: 저녁 22:00 GOOSE 앱 열어 오늘의 3줄 회고 |
| **Web** | 서연: 노트북에서 주간 회고 페이지로 학습 기록 검토 |
| **Desktop** | 민준: 백그라운드 실행, macOS 알림으로 리추얼 리마인드 |
| **CLI** | 민준: `goose morning` 입력 → 터미널에 오늘의 포커스 출력 |

### Tier 2: Daily (일상 확장)

| Channel | Scenario |
|---|---|
| **Telegram** | 지우: 점심시간 "방금 클라이언트가 이렇게 말했는데 속상해요" → 공감 + 재구성 |
| **Mobile** | 서연: 공부 중 모르는 개념을 GOOSE에게 질문, 저장됨 |
| **Web** | 민준: 주간 학습 로그를 GOOSE가 자동 정리한 페이지 확인 |
| **Desktop** | 지우: 캘린더 연동으로 오후 집중 타임 시작 알림 |
| **CLI** | 민준: `goose learn "channel select vs sync.Mutex"` 한 줄 로그 저장 |

### Tier 3: Social (관계 확장, v1.0 이후)

| Channel | Scenario |
|---|---|
| **Telegram** | 지우: 친구와 "우리 GOOSE 둘 다 3단계 됐대!" 공유 |
| **Mobile** | 서연: 가족 공유 리추얼 (자녀와 공부 시간 함께) |
| **Web** | 민준: GOOSE 커뮤니티에서 다른 사람들의 성장 스토리 열람 |

## 6. Audience Vocabulary (그들의 말)

### 지우가 자연스럽게 쓰는 말
- "요즘 루틴이 없어요"
- "아침이 너무 길어"
- "작심삼일"
- "혼자인 것 같아"
- "기록은 해야 하는데"

### 민준이 자연스럽게 쓰는 말
- "컨텍스트 스위칭", "리팩토링", "릴리즈"
- "한 줄 로그", "TIL"
- "마찰 없이"
- "자동화하고 싶은데"

### 서연이 자연스럽게 쓰는 말
- "꾸준히 하고 싶어요"
- "혼자 하니까 외로워요"
- "어려우면 못 해요"
- "애들 재우고"

→ 이 어휘들은 copywriting 시 hero/feature 카피의 기준이 됩니다.

## 7. Anti-Personas (타깃 아님)

- **Power user optimizer**: 초당 클릭을 줄이려는 사용자. GOOSE의 "느림"이 맞지 않음.
- **Team collaboration seeker**: 팀·공유 워크스페이스가 필요한 사용자. GOOSE는 1인칭 중심.
- **Novelty chaser**: 최신 AI 기능 체험 소비자. 꾸준한 관계 유지가 아닌 체험만 원함.

---

## 8. Accessibility Personas

- **시각 저하 사용자** — Telegram + CLI 채널로 스크린 리더 기본 호환
- **저사양 디바이스 사용자** — 모바일 APK size < 30MB 목표
- **저대역 네트워크** — 텍스트 우선, 이미지는 progressive
