# Research — SPEC-GOOSE-FORTUNE-001

## 1. Saju 계산 상세 알고리즘

### 1.1 기준점

- **갑자일(甲子日) 기준**: 1900-01-31을 갑자일 1로 잡고, 해당일로부터의 경과일 % 60 으로 일주 계산.
- 입춘(立春, 매년 2월 3-5일) 기준으로 년주 경계 결정. 양력 1월 또는 2월 초반생은 전년도 년주일 수 있음.

### 1.2 24절기 테이블

월주는 24절기의 "절기(節氣)" 기준. 월주 경계는 실제 태양황경에 따라 매년 ±1일 변동 → 정밀 계산 시 태양황경 공식 필요, 본 SPEC은 **평균값 테이블** 사용:

| 월 | 절기 | 평균일 | 월지(月支) |
|---|------|-------|---------|
| 1 | 立春 | 2/4 | 寅 |
| 2 | 驚蟄 | 3/6 | 卯 |
| 3 | 淸明 | 4/5 | 辰 |
| ... | ... | ... | ... |

### 1.3 오행(五行) 매핑

| 천간 | 오행 | 지지 | 오행 |
|-----|-----|-----|-----|
| 甲乙 | 木 | 寅卯 | 木 |
| 丙丁 | 火 | 巳午 | 火 |
| 戊己 | 土 | 辰戌丑未 | 土 |
| 庚辛 | 金 | 申酉 | 金 |
| 壬癸 | 水 | 亥子 | 水 |

8글자 중 가장 많은 오행 → `dominant`.

### 1.4 Goldenfile 검증

주요 역사 인물 및 특정 일자 표준 답과 비교:

| 날짜 | 일주 (기대) |
|------|----------|
| 1900-01-31 | 甲子 (기준) |
| 2000-01-01 | 戊午 |
| 2026-04-22 | 계산 필요 |

## 2. Biorhythm 수학

- **신체 (Physical)**: 23일 주기, `sin(2π × days / 23)`
- **감성 (Emotional)**: 28일, `sin(2π × days / 28)`
- **지성 (Intellectual)**: 33일, `sin(2π × days / 33)`

해석:
- +80% 이상: 고점 (peak)
- -20% ~ +20%: critical day (전환점, 주의)
- -80% 이하: 저점 (low)

과학적 근거 희박 → 엔터테인먼트 명시.

## 3. LLM Prompt 실험 (Mock)

### 3.1 Baseline

Without guard:
```
사용자: 1990-05-15 생, 오늘 일주 庚午, 감성 80%
LLM: "오늘은 주식 투자하기 좋은 날입니다! 확실히 수익을 낼 수 있어요..."
```
→ 금융 키워드 포함, 확정적 표현. **부적합**.

### 3.2 With Guard

Prompt에 다음 제약 추가:
- "투자/주식 언급 금지"
- "'반드시', '100%' 등 확정 표현 금지"
- "긍정적이지만 열린 표현 사용"

→ "오늘은 새로운 도전에 마음을 열어보면 좋을 것 같아요. 감성이 충만한 날이니, 오랫동안 미뤄둔 대화를 시도해보세요."

### 3.3 재생성 트리거

첫 응답 guard 통과율 LLM별 편차:
- Claude Haiku: ~95% 통과
- GPT-4o: ~93%
- Gemini Flash: ~88%

→ 3회 재생성 cap 충분.

## 10. 오픈 이슈

1. **월주·시주 정밀도**: 24절기 평균 테이블 vs 실시간 태양황경 계산. v0.1은 평균 테이블, 정확도 요구 시 astro library 후속 도입.
2. **사주 용신(用神) 해석**: 전통 명리학 핵심 개념이지만 고급. v0.1 범위 외.
3. **LLM이 한국어 운세 표현에 서툰 경우**: Few-shot examples를 프롬프트에 10개 정도 예시로 주입 (별도 fixture).
4. **생시 모르는 사용자**: 시주 없이 3기둥만 계산. 정확도 떨어지지만 경고 후 진행.
5. **윤달 처리**: 만세력 라이브러리 검증 필요.
